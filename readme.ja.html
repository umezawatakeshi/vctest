概要

Video Codec Test は、VCM 映像コーデックをテストおよびベンチマークするための Windows 用のツールです。

動作環境

必須環境

OS
Windows XP 以降

インストール等

適当なフォルダに全てのファイルを展開してください。アプリケーション自体はファイルシステムやレジストリに書き込まないので、展開したファイルを削除すればアンインストール完了です。

寄付について

Video Codec Test はフリーソフトウェアです。開発を支援するための寄付は常に歓迎しております。詳細は作者のblogの「寄付について」ページをご覧ください。
